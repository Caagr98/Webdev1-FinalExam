The invention of the Web

The Beginning

In the beginning, there was nothing.

Well, since it wasn't the absolute beginning, there was a world and all that, but nothing that matters.

Getting bored with this nothing, man created computers. And man saw that it was good.

Man was wrong, of course; those computers were nothing remarkable with today's standards.

However, at the time, today's standards were still the future, and with that today's standards, it was good.

That's beside the point though.

The computer was a neat invention, but then man decided to create the Internet, too.

The W3

After the beginning, it was fairly common to write scientific articles on computers, usually with various (nowadays primitive) markup languages with support for things like headings and italics.

CERN, for example, used a language called SGMLguid.

This language supported most things needed at the moment, but there was no way to link between different articles except simply saying the name and telling the reader to find it himself.

Therefore, in 1990, a CERN worker named Tim Berners-Lee (TimBL) invented the World Wide Web (originally abbreviated W3, now WWW), along with HTML and HTTP to implement it; the major feature of HyperText compared to other markup languages was the ability to HyperLink to other articles, creating a Web of documents.

The first recorded HTML document [1] is clearly identifiable as HTML, but the structure is completely different - instead of a tree, it appears to use a linear structure. Other early HTML documents [2] [3] also display this behavior, along with some non-standard tag names, most of which come from the previously mentioned SGMLguid.

The list of supported tags for this ancient version of HTML is archived here. Interestingly, TimBL expressed dissatisfaction at the <H{1..6}> tags, instead wanting a nested <SECTION> tag containing a generic <H> tag - one of which was added about 20 years later, while the other still doesn't exist.

Adoption

A few important points about the Web are that it's free and decentralized. Being free meant that anyone could use it for anything they want, and being decentralized means that there isn't one single server hosting the entire Web, which would have meant that the maintainer of that server would have total control over every website.

In January '93, there were about fifty web servers worldwide. By October the same year, there were over five hundred.

Early browsers include the original unnamed browser for the NeXT operating system, some unknown browser called Erwise and ViolaWWW, and the console-based Lynx (which still exists today, but it doesn't have very many users).

However, the browser that made the Web what it is today is Mosaic. Featuring an intuitive UI, a Windows version, and support for inline images (<img> tags), it quickly got popular. The same company developed another browser, known as Netscape Navigator, while Mosaic was licensed to Microsoft and made into Internet Explorer.
